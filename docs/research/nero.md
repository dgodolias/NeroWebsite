## Visual Asset Rule

Live Nero imagery must be real Nero imagery only. Generated scene exploration was removed from the project because it could misrepresent the venue.

Official sourced files live under:

- `public/assets/sourced/nero-official/`

Deterministic refined files live under:

- `public/assets/refined/`

Refinement is limited to crop, color/contrast, sharpening and WebP compression. No generative replacement is used for live venue/food photos.

## Transferable Principles From The Crema Rollout

- Research the actual business before designing.
- Use real or carefully sourced visual assets; never build a generic gradient landing page.
- Greek/local content first, with restrained English only where it works as hospitality language.
- Motion should have a job: reveal the venue, guide scroll, or make controls feel responsive.
- Test with Playwright, screenshots, and overflow checks before calling a milestone done.
